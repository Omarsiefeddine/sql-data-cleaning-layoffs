# SQL Data Cleaning Project

## Dataset
Layoffs dataset containing company name, date, percentage laid off, and stage.

## Objective
Clean raw data and make it ready for analysis.

## Cleaning Steps
- Removed duplicates using ROW_NUMBER()
- Standardized date format using STR_TO_DATE()
- Handled NULL and blank values
- Trimmed and normalized text fields

## Files
- layoffs_raw.csv → original dataset
- layoffs_cleaned.csv → cleaned dataset
- data_cleaning.sql → SQL cleaning script

## Skills Used
- SQL
- Data Cleaning
- Data Transformation
